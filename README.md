## Instruções dos Comandos GIT - Commits e Push
Segue abaixo as Intruções de Criação de Commits e Push:

Commit do Arquivo functions.php "Initial Commit":
```sh
$ git commit -m 'Initial Commit'
```

Commit de Edição do Arquivo functions.php "Editando o arquivo functions.php":
```sh
$ git commit -m 'Editando o arquivo functions.php'
```

Commit de Inserção do Arquivo README.md:
```sh
$ git commit -m 'Inserindo README.md'
```

Push para o Repositório Remoto do GitHub:
```sh
$ git push origin master
```
